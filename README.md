# Blinkit-PowerBI-Dashboard
PowerBI Dashboard Analyzing Blinkit sales and Delivery

## Overview
Interactive Power BI dashboard analyzing Blinkit's sales, orders, and delivery performance across cities and categories.

## Files
- `Blinkit_Dashboard.pbix` – Power BI file (open in Power BI Desktop).
- `Data/` – sample dataset (CSV/Excel).
- `Images/` – dashboard screenshots.

## Tools
Power BI Desktop, Power Query, DAX, Excel

## How to view
1. Download `Blinkit_Dashboard.pbix`.
2. Open in Power BI Desktop to interact with visuals.

## Notes
- Data is sample/sanitized; no confidential information included.
